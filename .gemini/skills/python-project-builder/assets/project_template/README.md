# Project Name

This is a boilerplate Python project, managed by the `python-project-builder` Gemini CLI skill.

## Setup

1.  Create a virtual environment: `python -m venv .venv`
2.  Activate the environment: `source .venv/bin/activate`
3.  Install dependencies: `poetry install` (if using poetry) or `pip install -e .`

## Development

Write your application code in the `src/` directory.

## Testing

Run tests using pytest: `poetry run pytest`

## Linting & Formatting

Check code style with ruff and black: `poetry run ruff check .` and `poetry run black .`
